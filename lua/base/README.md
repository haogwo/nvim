# Base 快捷键

核心编辑器快捷键配置。

## Leader 键
`<Space>` - 主 Leader 键

## 窗口操作

| 快捷键 | 功能 | 模式 |
|--------|------|------|
| `<leader>h` | 跳转到左侧窗口 | n |
| `<leader>j` | 跳转到下方窗口 | n |
| `<leader>k` | 跳转到上方窗口 | n |
| `<leader>l` | 跳转到右侧窗口 | n |
| `<leader>wk` | 增加窗口高度 | n |
| `<leader>wj` | 减少窗口高度 | n |
| `<leader>wh` | 减少窗口宽度 | n |
| `<leader>wl` | 增加窗口宽度 | n |
| `<leader>w=` | 均分所有窗口大小 | n |

## 编辑操作

| 快捷键 | 功能 | 模式 |
|--------|------|------|
| `J` | 向下移动选中行 | v |
| `K` | 向上移动选中行 | v |
| `Y` | 复制到系统剪贴板 | v |

## 搜索

| 快捷键 | 功能 | 模式 |
|--------|------|------|
| `<leader>nh` | 清除搜索高亮 | n |
| `n` | 下一个匹配项（保持居中） | n |
| `N` | 上一个匹配项（保持居中） | n |
