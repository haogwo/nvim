# Plugins 快捷键

各插件的快捷键配置。

## Leader 键
`<Space>` - 主 Leader 键

## LSP (lsp.lua)
*仅在 LSP 附加到缓冲区时生效*

| 快捷键 | 功能 |
|--------|------|
| `K` | 悬停查看文档 |
| `gd` | 跳转到定义 |
| `gr` | 查看引用 |
| `<leader>ca` | 代码快速修复 |
| `<leader>rn` | 重命名符号 |
| `<leader>e` | 显示当前行诊断信息 |

## Telescope (telescope.lua)

| 快捷键 | 功能 |
|--------|------|
| `<leader>ff` | 查找文件 |
| `<leader>fg` | 实时搜索 |
| `<leader>fb` | 缓冲区列表 |
| `<leader>fh` | 帮助标签 |

## Markdown (markdown.lua)
*仅在 markdown 文件类型时生效*

| 快捷键 | 功能 |
|--------|------|
| `<leader>mv` | 切换 Markdown 预览 |
| `<leader>ms` | 切换分屏预览 |
| `<leader>mt` | 切换目录 (TOC) |

## Outline (ui.lua)

| 快捷键 | 功能 |
|--------|------|
| `<leader>o` | 切换代码大纲 |
